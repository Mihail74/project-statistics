# project-statistics

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:12222
npm run dev
```

## Production Setup

``` bash
# install dependencies
npm install

# build for production with minification
npm run build

# install pm2
npm i pm2 -g

# start by pm2
pm2 start ecosystem.config.js --env production
```
